# Floor Plan Tool MVP - Slides Outline

## Slide 1 - Problem
- Event teams need fast room setup planning.
- Current process is fragmented (spreadsheets, static diagrams, manual redraws).
- Lack of a quick visual tool slows planning and approvals.

## Slide 2 - Current Pain
- Template setup is repetitive for common event formats.
- Asset placement is manual and inconsistent.
- Teams struggle to communicate floor changes quickly.

## Slide 3 - Solution (MVP)
- Single `/floorplan` UI for rapid mock layouts.
- Three starter templates: Banquet, Theater, Cocktail.
- Furniture palette with core venue assets.
- Simple selection controls: add, nudge, rotate, delete.
- Dock venue background overlay for context.

## Slide 4 - Demo Flow
- Open `/floorplan`.
- Switch between templates to show instant layout presets.
- Add assets from palette.
- Select one object, nudge and rotate it, then delete it.
- Show dock overlay fallback behavior if image is missing.

## Slide 5 - Next Steps
- Add drag-and-drop and resize handles.
- Save/load layouts to Supabase.
- Multi-user collaboration and comments.
- Export/print layouts for operations teams.
- Expand template and venue catalog.
